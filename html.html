Basic tags

<html></html> Creates an HTML document
<head></head> Sets off the title & other info that isn't displayed
<body></body> Sets off the visible portion of the document
<title></title> Visibile in browser title bar; when bookmarking pages, this is what is bookmarked

Text tags

<pre></pre> Creates preformatted text
<h1></h1> Creates headlines - H1 (largest) to H6 (smallest)
<em></em> Emphasises a word (usually in italics)
<strong></strong> Emphasises a word (usually in bold)

Links

<a href="URL">clickable text</a> Creates a hyperlink to a URL
<a href="mailto:EMAIL_ADDRESS">clickable text</a> Creates a hyperlink to an email address
<a name="NAME"> Creates a target location within a document
<a href="#NAME">clickable text</a> Creates a link to that target location

Formatting

<p></p> Creates a new paragraph
<br> Inserts a line break (carriage return)
<blockquote></blockquote> Puts content in a quote - indents text from both sides
<div></div> Used to format BLOCK content with CSS
<span></span> Used to format INLINE content with CSS

Lists

<ul></ul> Creates an unordered list
<ol start=?></ol>ol> Creates an unordered list (start=xx where x is a counting number)
<li></li> Encompasses each list item

Graphical elements

<hr> Inserts a horizontal rule
<hr size=?> Sets height of horizontal rule
<hr width=?> Sets width of rule (as a % or absolute pixel length
<hr noshade> Inserts a rule with no shadow
<img src="URL" /> Adds image; it is a separate file located at URL
<img src="URL" height=? width=?> Set the height and width of the image, in pixels
<img src="URL" alt=?> Sets the alternate text to be displayed if the browser (or user) can't process images

Forms

<form></form> Defines a form
<select multiple name=? size=?></select> Creates a scrolling menu. Size sets the number of menu items visible before user needs to scroll.
<option> Sets off each option
<textarea name=? cols="x" rows="y"></textarea> Creates a text box area. Columns set the width; rows set the height
<input type="checkbox" name=? value=?> Creates a checkbox
<input type="checkbox" name=? value=? checked> Creates a checkbox which is pre-checked
<input type="radio" name=? value=?> Creates a radio button
<input type="radio" name=? value=? checked> Creates a radio button which is pr-checked
<input type="text" name=? size=?> Creates a one-line text area. Size sets length, in characters
<input type="submit" value=?> Creates a submit button. Value sets the text in the submit button
<input type="image" name=? src=? border=? alt=?> Creates a submit button using an image
<input type="reset"> Creates a reset button.
<input type="email" name=?> Creates a single-line textbox for email addresses
<input type="url" name=?> Creates a single-line textbox for URLs
<input type="number" name=?> Creates a single-line textbox for a number
<input type="range" name=?> Creates a single-line textbox for a range of numbers
<input type="date/month/week/time" name=?> Creates a single-line textbox with a calendar showing date/month/week/time
<input type="search" name=?> Creates a single-line textbox for searching
<input type="color" name=?> Creates a single-line textbox for picking a color

Tables

<table></table> Creates a table
<tr></tr> Creates a row in a table
<td></td> Creates a cell in a row
<th></th> Creates the table header (a normal cell with bold, centered text

Table attributes

<table border=?> Sets the width of the border around table cells
<table cellspacing=?> Sets amount of space between cells
<table cellpadding=?> Sets aount of space between a cell's border and its contents
<table width=?> Sets width of the table in pixels or as a percentage
<tr align=?> Sets alignment for cells within the row (left/center/right)
<td align=?> Sets alignment for the contents of cells (left/center/right)
<tr valign=?> Sets vertical alignment for cells within the row (top/middle/bottom)
<td valign=?> Sets vertical alignment for the contents of cells (top/middle/bottom)
<td rowspan=?> Sets number of rows a cell should span (default=1)
<td colspan=?> Sets number of columns a cell should span
<td nowrap> Prevents lines within a cell being broken to fit
